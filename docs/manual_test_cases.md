# Тестовая документация для ручных тестов сервиса F‑Bank

**Версия:** 1.0  
**Дата:** 2026‑04‑25  
**Автор:** Злобина Софья  

**Цель тестирования:** выявить дефекты в функциональности перевода средств, валидации входных данных и расчёте комиссии.

**Объект тестирования:** веб‑сервис F‑Bank (локальный запуск: `http://localhost:8000/`).

**Окружение:**
* браузеры: Edge (последние версии);
* ОС: Windows 11;
* разрешение экрана: 1920×1080.

|--------|---------|
| **ID** | TC‑001 |
| **Приоритет** | Major |

| **Фактический результат** | Перевод на 0 ₽ принимается, появляется уведомление о принятии запроса. |
| **Постусловия** | Отменить незавершённый перевод (если возможен). |
| **Комментарии** | Дефект влияет на логику бизнес‑процесса — позволяет нецелевые операции. |

| **Постусловия** | Зафиксировать успешный перевод в логах. |
| **Комментарии** | Ключевой рабочий сценарий — основа пользовательского опыта. |
